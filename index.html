Installation and Setup

  • Install the latest version of Python 3

  • Install Brownie

    python3 -m pip install --user pipx
python3 -m pipx ensurepath

    May need to restart terminal before the next steps

    pipx install eth-brownie
pip install py-solc-x
pip install python-dotenv

  • Get 0.1 testnet Goerli ETH and 50 testnet LINK from a faucet

Running the code

  • Clone https://github.com/SaileshBechar/Resonance

  • In the project directory, create a .env file that contains:

    PRIVATE_KEY=your wallet private key here

  • Create an API key at Infura and add it to the .env file

    WEB3_INFURA_PROJECT_ID=your infura api key here

  • Run the experiments from the project directory

    brownie test -v -s --network goerli

    Alternatively, just run the singular unit test 

    brownie test -v -s --network goerli -k test_can_create_and_cleanup_collectible

    and the Popularity test

    brownie test -v -s --network goerli -k test_popularity
